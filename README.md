# ConsoleReversi

You can play the reversi in your terminal!!

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'console-reversi'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install console-reversi

## Start Game

```
$ console-reversi
```

## Commands

* move a cursor : `direction keys`
* put a piece : `Enter`
